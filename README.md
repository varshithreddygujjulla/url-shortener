# URL Shortener 🔗

A backend application that converts long URLs into short, shareable links.

## Features
- Generate shortened URLs
- Redirect users to the original URL
- Track number of visits (analytics)
- Store URLs in a database

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

1. Clone the repository

git clone https://github.com/varshithreddygujjulla/url-shortener.git

2. Install dependencies

npm install

3. Start the server

node app.js
